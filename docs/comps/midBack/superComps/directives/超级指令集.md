# 超级指令集

## v-ellipsis
* > 文本超出省略号效果
```html
 <!-- 直连查询文本展示块 -->
<div
  v-if="intoItem.hasOwnProperty('value')"
  @mouseenter="(e) => setTip(e, 1)"
  @mouseleave="(e) => hiddenTip(e, 1)"
>
  <span v-ellipsis.100 v-if="intoItem.isEllipsis">
    {{ intoItem.value }}
  </span>
  <span v-else>{{ intoItem.value }}</span>
</div>
```
